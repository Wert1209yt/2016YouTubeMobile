## Instalation
1. `git clone https://github.com/Wert1209yt/2016YouTubeMobile`
2. `cd 2016YouTubeMobile`
3. Set up config.php
4. `node server.js`
### Shutdown
<kbd>CTRL</kbd> + <kbd>C</kbd> - *Server will shutdown*
